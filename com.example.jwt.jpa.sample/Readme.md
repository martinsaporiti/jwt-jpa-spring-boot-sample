#  Spring Boot JWT + JPA Sample

### Intro
Este proyecto pretende ser la base para futuros desarrollos en Java con Spring Boot que requieran de una API segurizada 
utilizando JWT.
Este template se apalanca en las siguientes tecnologías:
* Spring Boot
* Spring Security
* JWT
* JPA
* MYSQL

El modelo utilizado para la autenticación y autorización no es el que provee gratias Spring Security. 
Se utiliza un modelo custom.
